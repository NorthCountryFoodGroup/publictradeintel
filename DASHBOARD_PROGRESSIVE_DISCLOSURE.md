# Dashboard Progressive Disclosure

Version 2.2 Sprint 1 reorganizes the Dashboard so it acts more like a command center.

Always visible:

- AI Market Brief.
- Run Prediction Scan.
- Today's Opportunities.

Collapsible sections:

- Prediction Scan Details.
- Market Overview.
- AI Prediction Engine Details.
- Watchlist Summary and Congress Activity.
- Policy Activity and Prediction Performance.

Expanded/collapsed state is stored in local storage so the dashboard remembers the user's preference.
