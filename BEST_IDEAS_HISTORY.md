# Best Ideas History

Best Ideas history is seeded in Version 2.2 Sprint 2 so future scans can show stability and movement.

## History Fields

- daysOnBestIdeas
- newThisScan
- returnedToList
- movedUp
- movedDown
- additions
- removals
- archive rows by scan

The current implementation publishes the history structure with each completed scan. Longer-term performance grading remains part of the prediction performance roadmap.
