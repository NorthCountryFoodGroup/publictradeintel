# Category Rankings

Stocks to Buy category rankings are independent lists built from all qualified analyzed stocks.

They are not created by filtering the Overall Top 25.

## Price Boundaries

- $5 and under: `price > 0 && price <= 5`
- $5.01-$10: `price > 5 && price <= 10`
- $10.01-$25: `price > 10 && price <= 25`
- $25.01-$100: `price > 25 && price <= 100`
- Over $100: `price > 100`

## List Matrix

The center publishes six price categories across four timeframes, for 24 ranking lists per completed scan.

Every list preserves:

- qualified count
- displayed count
- average unified score
- average confidence
- average risk
- leading sector
- fresh data percentage
- reference scan timestamp

Standards are not weakened to fill 25 names.
