# Bull / Bear Methodology

Bull Case and Reasons to Wait are generated only from fields already present in the selected prediction record.

## Bull Case Sources

- Multi-timeframe alignment.
- EMA/VWAP technical fields.
- Confirmed setup signals.
- Chart pattern direction.
- Relative volume.
- Rank or score improvement.
- Technical score.
- Risk/reward ratio.
- Congressional activity as a secondary catalyst.
- Data quality.
- Stored strongest signals.

## Reasons To Wait Sources

- Near resistance.
- Weak relative volume.
- Mixed alignment.
- Stale or failed data quality.
- Missing critical fields.
- Forming or failed setup.
- Bearish chart pattern.
- Elevated risk score.
- Unfavorable risk/reward.
- Stored conflicting signals.
- Negative policy/news signals.

Unavailable signals are not treated as negative. They are listed separately in Signal Agreement.
